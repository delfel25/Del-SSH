# READ BEFORE INSTALLATION

**CONFIG FILE IN ~/.ssh_servers**

```
nano .ssh_servers
vim .ssh_servers
nvim .ssh_servers 
```

*All your servers and other things that you specified are added there.*

# INSTALLATION
```
git clone https://github.com/delfel25/Del-SSH.git
cd Del-SSH
chmod +x DelSSH.sh
./DelSSH.sh
```
